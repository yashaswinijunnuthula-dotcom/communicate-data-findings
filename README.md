# Communicate Data Findings

## Project Overview

This project analyzes the Diamonds dataset using exploratory and explanatory data analysis.

## Dataset

The dataset contains 53,940 diamonds with features such as:

- Carat
- Cut
- Color
- Clarity
- Depth
- Table
- Price
- x
- y
- z

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Files

- Part_I_Exploration.ipynb
- Part_I_Exploration.html
- Part_II_Explanatory.ipynb
- Part_II_Explanatory.html

## Key Findings

- Carat has the strongest influence on diamond price.
- Higher clarity generally increases price.
- Better color grades increase price.
- Cut also affects price but less than carat.
