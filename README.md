# Titanic Data Analysis - Communicate Data Findings

## Project Overview

This project is part of the Udacity Data Analyst Nanodegree.

The objective of this project is to perform exploratory data analysis (EDA) on the Titanic dataset and communicate the most important findings through an explanatory presentation.

## Dataset

The Titanic dataset contains information about passengers aboard the RMS Titanic.

The dataset includes:

- Passenger Class
- Age
- Gender
- Fare
- Embarkation Port
- Family Information
- Survival Status

**Total Records:** 891

**Total Features:** 15

## Key Findings

- Most passengers traveled in Third Class.
- Female passengers had higher survival rates.
- Higher fares were associated with higher survival.
- Passenger class strongly influenced fare.
- Age alone had a weak relationship with survival.
